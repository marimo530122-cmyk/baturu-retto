# バツルーレット 要件定義書（v1.0）

作成日: 2026年7月6日

---

## 1. ゲームの概要

| 項目 | 内容 |
|---|---|
| ゲーム名 | **バツルーレット**（英語名: **Batsu Roulette**） |
| 一言でいうと | 飲み会・パーティーの「盛り上げ役（主催者）」を助ける、罰ゲーム付きルーレットゲーム |
| 遊ぶ場面 | 飲み会、合コン、パーティー、宅飲みなど、お酒の席 |
| 遊ぶ人 | 男女混合の複数人（男3・女3など）。男性だけ・女性だけでもOK |
| 目標 | 無料版を全世界に配信し、有料版（買い切り）で収益化する |

※名前は「短くて覚えやすい」「何のゲームか伝わる」「海外でもそのまま通じる（Batsu Gameは海外のお笑いファンに知られている言葉）」の3点で選びました。

---

## 2. 遊び方（基本の流れ）

スマホ**1台**をテーブルの真ん中に置いて遊びます。

1. 参加者の名前を登録する（「男女に分ける」「全員一緒」のどちらかを選べる）
2. 「ルーレットスタート」ボタンを**1回押すだけ**で、ルーレットが回って
   「やる人」と「相手」が決まり、お題まで一気に発表される（**ワンクリック完結**）
3. お題は**2,500通り（シチュエーション50種 × 行動50種の掛け算）**から生成され、
   画面に大きく表示 ＋ 司会者風の音声で読み上げられる
4. お題を実行！ 盛り上がったら、また次のルーレットへ

- **👑 王様モード**：10%の確率でルーレットが「王様」を選ぶ。王様は自分で自由にお題（命令）を出せる
- **パス機能**：出たお題が難しければ、1タップで別のお題に交換できる（無理強いしない設計）

## 3. 無料版と有料版

### 無料版（世界中の誰でも遊べる入り口）
- 通常お題パック（一部、例：500通り分）
- 男女混合・複数人モード
- **🎷 ジャズバー風BGM**（無料版に含めることを決定：2026年7月6日）
- 声のキャラクター切り替え（🎤MC／👨渋いおやじ／👧かわいい女子／🎲おまかせ）
- 小さめの広告を表示

### 有料版（2026年7月7日 内容確定）

**売り方：買い切り1回（480円）で全部解除**（複数パックのバラ売りはしない。理由：飲み会の場で「これは有料？」と迷う瞬間をなくし、即決してもらうため）

| # | 機能 | 内容 |
|---|---|---|
| 1 | **お題 全2,500通り** | 無料版は一部（例：500通り）のみ。有料で全解放 |
| 2 | **🔞 大人向けパック** | セクシー・ちょい過激なお題。18歳以上の確認画面つき |
| 3 | **👨‍👩‍👧 ファミリー向けパック** | お酒なしでも家族・子どもと遊べるお題 |
| 4 | **💑 1対1モード** | 男性1人×女性1人のサシ飲み・カップル用 |
| 5 | **広告の非表示** | 無料版の小さな広告を消す |
| 6 | **📮 お題の投稿・共有機能** | 課金ユーザー同士で自作のお題を投稿し合い、遊び合える（開発の後半ステップで追加。投稿には通報ボタン＋NGワードチェックを必須セット） |
| 7 | **🃏 イカサマモード（幹事専用）** | 幹事だけがこっそり使える裏機能。「次のルーレット、この人に当たりやすくする」を設定できる。誕生日の主役やいじられ役のエースに仕込める。表向きは完全ランダムに見えるので座が盛り上がる |
| 8 | **🌶️ 激辛レベル調整（ダイヤル）** | お題の過激さをレベル1（ほんわか）〜5（地獄）で調整。無料版はレベル2固定、有料版で全開放。大人向けパックとも連動 |
| 9 | **🎨 ルーレットの着せ替え** | 金ピカカジノ風・和風居酒屋風など、見た目のテーマを切り替え |

**将来候補（今回は仕様に明記のみ、実装は別枠）**
- **🎙️ プロ声優ボイス**：本格的な収録ボイス。声優への依頼・収録費用が発生するため、有料版本体とは別の追加コンテンツ（都度課金など）として検討

**売り方のポイント**：
- タイトル画面では無料の「はじめる」ボタンを**一番大きく・光らせて**目立たせ、まず遊んでもらうことを最優先にする
- 有料メニューのボタンも最初から🔒つきで見えていて、タップすると「有料版のご案内」が出る形にする（欲しくなる瞬間を逃さない仕組み）

## 4. 見た目（デザイン）

- **ネオン・夜の街風**：黒背景にピンク・紫のネオンの光。大人の夜の雰囲気
- ルーレットと文字は**大きく**（酔っていても読める・押せる）
- ボタンは少なく・大きく、操作は基本**1タップ**

## 5. 言語展開（全世界配信の順番）

| 順番 | 言語 | 理由 |
|---|---|---|
| 第1弾 | 日本語 | まずここで完成させる |
| 第1弾 | 英語 | 世界共通語。これだけで広い層に届く |
| 第2弾 | 韓国語 | 世界有数の飲み会ゲーム文化の国 |
| 第2弾 | 中国語（繁体字・簡体字） | 台湾・香港の飲み会文化＋世界最大級の話者数 |
| 第3弾 | スペイン語 | 中南米・スペインのパーティー文化圏 |

**重要**：翻訳は直訳ではなく、**各国の飲み会ネタに合わせてお題を作り直す**（例：「昭和のヤンキー風」→ 英語圏なら通じるネタに置き換え）。
国ごとの配慮ルールとサンプルは **`世界お題ガイド.md`** にまとめてある（全世界共通の禁止事項6か条＋英語圏・韓国・中華圏・スペイン語圏のお題例）。

## 6. 使う道具（かんたん説明つき）

| 道具 | 何をするもの？ | 費用 |
|---|---|---|
| HTML / CSS / JavaScript | ブラウザで動くゲームを作る基本の材料。**お題生成部分はすでに完成済み** | 無料 |
| Web Speech API | スマホに最初から入っている「読み上げ機能」を使う仕組み | 無料 |
| Vercel または Netlify | 作ったゲームを世界中に公開してくれる置き場所 | 無料から |
| Capacitor（後半で） | ブラウザ版をそのままスマホアプリに変身させる道具。ストア経由なら480円の課金や年齢制限の仕組みをストアが代行してくれる | 無料（ストア登録料は別途） |
| Firebase（後半で） | お題の投稿・共有に必要な「データの保管庫」 | 小規模なら無料から |

## 7. 安心・安全のルール

- 大人向けパックは**年齢確認画面**を必ず通す
- **パス機能**でお題を強制しない
- **飲酒を強要するお題は入れない**（「20歳未満の飲酒は法律で禁止」の注意書きを表示）
- 投稿されたお題には**通報ボタンとNGワードチェック**をセットで用意する

  ルーレット画面＋名前登録＋お題2,500通り＋音声読み上げ＋ネオンデザイン

  お題の投稿・共有機能（Firebase）→ 韓国語・中国語・スペイン語版

## 9. すでにできているもの（2026年7月7日 時点）

- `index.html` / `style.css` / `game.js` — ゲーム本体（ネオンデザイン、ワンクリックルーレット、👑王様モード10%、5言語切り替え、有料版ご案内ボタン、🏆10ラウンドごとの表彰式）
- `odai-generator.js` — お題の生成（日本語・英語・繁體中文・한국어・Español、各2,500通り）＋読み上げ最適化＋声のキャラクター対応
- `bgm.js` — 🎷ジャズバー風BGM（コードで自動演奏するので著作権フリー。読み上げ中は自動で音量が下がる）
- 声のキャラクター切り替え — 🎤標準MC／👨渋いおやじ（男性の声・低音）／👧かわいい女子（女性の声・高音）／🎲おまかせ（毎回変わる）
- **🎨 ルーレットの着せ替え（有料機能）** — 🌃ネオン／🎰カジノ（金・赤・黒）／🏮居酒屋（暖色）の3テーマ。CSS変数で色を一括管理しており、ルーレットの盤面色も連動する
- **🃏 イカサマモード（有料機能・幹事専用）** — ゲーム画面から次の1回だけ、狙った人が70%の確率で当たるように仕込める。使うと自動で1回限り解除される
- **演出のブラッシュアップ（2026年7月7日）** — `sfx.js`（スピン開始・お題発表・王様誕生の効果音）／`confetti.js`（お題発表・王様誕生の瞬間に舞う紙吹雪、王様は金色）／スピン中の煽りテキスト点滅／当たったルーレットのコマが白くフラッシュ／背景のうっすらしたグレイン質感／結果発表時のスマホ振動（対応端末のみ）
- `公開用` フォルダ — 全世界公開のときにアップロードするファイル一式
- `test.html` — 動作確認用テストページ

※声は端末に入っている読み上げ音声を加工して演出している。本格的な収録ボイス（声優ボイス）は有料版の目玉候補。
※有料機能は、アプリストア課金（ステップ3）が入るまでの開発中は `?premium=1` をURLに付けて動作確認できる（本番では通常の480円課金で解放）。
※🌶️激辛レベル調整は、大人向けパックのお題データがまだ存在しないため未実装。次に着手する候補。
