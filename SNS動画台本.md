# バツルーレット SNS動画台本集

作成日: 2026年7月18日

TikTok・Instagramリール・YouTube Shorts向けの「操作説明動画」の構成案と、セットで使う投稿キャプションをまとめる。1エントリー＝1投稿分。

---

## Day1：基本の操作方法編

### 動画の狙い
「難しい準備なしで、スマホ1台・ボタン数回で盛り上がる」を最速で伝える。チュートリアル感を出しすぎず、テンポよく「実際に盛り上がっている様子」に見せる。

### 基本情報
- 長さ目安：**15〜22秒**（縦型 9:16、TikTok/リール/Shorts共通）
- BGM／効果音：アプリ内蔵の🎷ジャズBGM・スピン音・読み上げ音声をそのまま収録に使う（端末の音量をオンにして撮影。ミュートで撮ると一番の魅力が伝わらないので注意）
- 撮影のコツ：既存機能の [inapp-browser-banner等の案内は出ないよう] Chromeなど通常ブラウザで開き、F11相当のフルスクリーン表示（アドレスバーが映り込まないよう）で撮影するときれい

### ステップ構成（画面操作 × 見せ方）

| 時間目安 | 画面操作 | 見せ方・画面内テキスト（テロップ） |
|---|---|---|
| 0:00〜0:02 | タイトル画面をアップで映す（ネオンのロゴが光っている状態） | テロップ：「まだ普通の飲み会してるの？」 |
| 0:02〜0:05 | 「🎰 はじめる」をタップ → 名前入力画面で2〜3人分の名前をテンポよく入力（早回し可） | テロップ：「①名前を入れるだけ」 |
| 0:05〜0:09 | 「🎲 ゲームスタート！」→ ルーレット画面。「🎰 ルーレット スタート！」をタップし、ホイールが回転する様子をしっかり見せる | テロップ：「②ボタン1つで運命の相手が決定」 |
| 0:09〜0:14 | 当たった瞬間のネオン演出＋紙吹雪＋お題カード表示。音声読み上げが鳴っている状態をそのまま収録（字幕でお題本文も表示するとミュート視聴者にも伝わる） | テロップ：「③お題が音声付きで発表される🔥」 |
| 0:14〜0:18 | （撮れ高が良ければ）👑王様モード発動の瞬間。金色の紙吹雪演出 | テロップ：「10%の確率で"王様"爆誕👑命令は絶対」 |
| 0:18〜0:22 | 「📤 シェア」ボタンをタップする動作 → タイトル画面のロゴに戻る | テロップ：「結果はワンタップでシェアできる／無料で今すぐ遊べる」 |

**撮り直しのポイント**：0:09〜0:14のお題発表の瞬間が動画の一番の見せ場。ここで面白いお題（例：ものまね系・褒め合い系など笑いが起きやすいもの）が出るまで何回か回してから本番を撮ると、初見の人の食いつきが良くなる。王様モードは10%なので毎回は出せなくてOK（出た回をラッキー素材として保存しておき、別の投稿にも使い回せる）。

**本文：**

```

2,500通り出てくるルーレット見つけた…

✅ 名前を入れるだけで即スタート
✅ お題は音声で読み上げてくれるから自分で言わなくてOK
✅ 10%の確率で"王様"が爆誕して命令し放題👑
✅ 結果はワンタップでシェアできる

次の飲み会、これで確定しました🎰
無料で今すぐ遊べます → プロフィールのリンクから🔗

#バツルーレット #BatsuRoulette #飲み会ゲーム #罰ゲーム
#宅飲み #合コン #大学生の日常 #パリピ #王様ゲーム #スマホゲーム
```

**冒頭フックの差し替え案（A/Bテスト用）：**
- 「王様ゲームより盛り上がるって噂のやつ、試してみた」
- 「幹事の準備0秒で盛り上がる飲み会ゲーム見つけた」
- 「このアプリ、飲み会の空気変えすぎた」
